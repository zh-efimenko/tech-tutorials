# Установка MCP серверов для Claude Code

## Обзор серверов

### Context7
Подтягивает актуальную документацию библиотек и фреймворков прямо в контекст разговора — из официальных репозиториев, не из обучающих данных модели.

**Когда использовать:**
- Спрашиваешь про API конкретной версии библиотеки (Spring Boot, React, Prisma и т.д.)
- Нужны примеры кода под конкретную версию
- Подозреваешь, что знания модели устарели

**Как использовать:**
Просто упомяни название библиотеки в вопросе — Claude сам вызовет Context7:
```
Как настроить @KafkaListener в Spring Boot 4?
Покажи пример useQuery из React Query v5
```

---

### Playwright
Управляет реальным браузером (Chromium) — может открывать страницы, кликать, заполнять формы, делать скриншоты, перехватывать сетевые запросы.

**Когда использовать:**
- Автоматизация действий в браузере
- Тестирование UI вручную или через Claude
- Скрейпинг страниц, требующих JS-рендеринга
- Отладка веб-интерфейсов

**Как использовать:**
Дай задачу на браузерное действие:
```
Открой https://example.com и сделай скриншот
Заполни форму логина на сайте X, логин admin, пароль 123
Перейди на страницу и скопируй текст из таблицы
```

---

### Exa
Поисковик нового поколения с семантическим поиском и возможностью читать полный контент страниц. В отличие от обычного поиска — понимает смысл запроса, а не только ключевые слова.

> **Платный сервис.** Exa тарифицирует за запросы, но выгоднее альтернативы: когда Claude ищет информацию сам (через WebSearch/WebFetch), он тратит в разы больше токенов на обработку и суммаризацию страниц. Exa возвращает чистый результат — меньше токенов, меньше денег за сессию.

**Когда использовать:**
- Нужна свежая информация (новости, changelogs, релизы)
- Поиск по смыслу, а не по точному совпадению слов
- Нужно прочитать содержимое конкретного URL
- Исследование темы по нескольким источникам

**Как использовать:**
```
Найди последние breaking changes в Spring Boot 4
Что нового в Java 24?
Прочитай содержимое этой страницы: https://...
Найди статьи про паттерн Saga в микросервисах
```

## Предварительные требования

| Требование | Для чего | Как проверить |
|---|---|---|
| Claude Code | Базовое требование | `claude --version` |
| Node.js 18+ | Playwright (запуск через `npx`) | `node --version` |
| Exa API Key | Аутентификация для Exa | Получить на https://exa.ai → Dashboard → API Keys |

Context7 и Exa — удалённые HTTP-серверы, локальных зависимостей не требуют.
Playwright запускается локально через `npx`, поэтому нужен Node.js.

## Установка

```bash
# 1. Context7 — документация библиотек в реальном времени
claude mcp add context7 --transport http https://mcp.context7.com/mcp

# 2. Playwright — автоматизация браузера
claude mcp add playwright -- npx @playwright/mcp@latest

# 3. Exa — веб-поиск и чтение страниц
claude mcp add exa --transport http "https://mcp.exa.ai/mcp?exaApiKey=ВСТАВЬ_СВОЙ_EXA_API_KEY"
```

## Проверка

```bash
claude mcp list
```

Должно показать все 3 сервера со статусом `Connected`.

## Получение Exa API Key

1. Зайди на https://exa.ai
2. Зарегистрируйся / войди
3. Перейди в Dashboard -> API Keys
4. Создай ключ и подставь в команду выше вместо `ВСТАВЬ_СВОЙ_EXA_API_KEY`
